# 📋 Project Backlog: Новая логика спавна квестовых машин (CONCEPT.md)

## 🎯 Проблема

Текущая реализация квестовых машин удаляет их при уходе за экран (через 5-8 секунд), что противоречит новой концепции:
**Машины не должны удаляться. Новые машины должны спавниться каждые 5-15 секунд.**
